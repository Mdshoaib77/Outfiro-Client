## About the Project

**Outfiro** is a full-stack MERN fashion eCommerce platform where users can browse and purchase trendy clothing for **Men, Women, and Kids**.

The platform includes a **separate Admin Panel** for managing products, orders, users, and inventory.

It provides a modern UI with smooth animations using **Framer Motion** and a responsive design built with **Tailwind CSS**.

---

## Project Overview

Outfiro aims to provide a modern online fashion shopping experience with fast performance and an easy-to-use interface.

* Build a scalable **MERN stack eCommerce platform**

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/outfiro-fashion-store
cd outfiro-fashion-store
```

### 2. Install dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

---

### 3. Create environment variables

Create a `.env` file in the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 4. Run the project

Backend:

```bash
npm run server
```

Frontend:

```bash
npm run dev
```
